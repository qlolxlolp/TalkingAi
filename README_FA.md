# 🏢 سامانه هوشمند تماس صوتی خودکار - شرکت توزیع نیروی برق استان ایلام
## نسخه عملیاتی دسکتاپ (Standalone EXE)

### 🎯 ماموریت سیستم
این سامانه به عنوان **"اپراتور هوشمند وصول مطالبات"** و **"پاسخگوی خدمات مشترکین"** برای شرکت توزیع نیروی برق شهرستان ایلام طراحی شده است. هدف اصلی، تسهیل فرآیند اطلاع‌رسانی بدهی‌ها، دریافت وعده‌های پرداخت و پاسخگویی ۲۴ ساعته به مشترکین با استفاده از هوش مصنوعی پیشرفته و کاملاً مستقل است.

---

### ✨ ویژگی‌های کلیدی

#### 1. هسته VoIP کاملاً نرم‌افزاری و مستقل
- **بدون نیاز به سخت‌افزار تلفن**: پیاده‌سازی کامل پروتکل‌های SIP/RTP در سطح کد
- **تماس خروجی و ورودی خودکار**: قابلیت راه‌اندازی کمپین‌های انبوه تماس
- **مدیریت همزمان چندین خط**: مقیاس‌پذیری برای هزاران تماس روزانه

#### 2. هوش مصنوعی مکالمه‌گر فارسی (Human-Like)
- **درک طبیعی گفتار (NLU)**: تحلیل نیت کاربر بدون وابستگی به کلمات کلیدی ساده
- **پاسخ‌های پویا و غیرتکراری**: تولید جملات متنوع بر اساس contexte مکالمه
- **حافظه مکالمه**: به خاطر سپردن اطلاعات رد و بدل شده در طول تماس
- **لحن انسانی و محترمانه**: تنظیم لحن بر اساس وضعیت مشترک (بدهکار، عادی، ویژه)

#### 3. اتصال بلادرنگ به داده‌های واقعی
- **متصل به سامانه eserv.bargh-ilam.ir**: دریافت لحظه‌ای اطلاعات بدهی، مصرف و وضعیت انشعاب
- **ثبت آنی وعده‌های پرداخت**: ارسال مستقیم داده‌ها به دیتابیس مرکزی شرکت
- **همگام‌سازی دوطرفه**: هر تغییر در سامانه اصلی بلافاصله در مکالمه منعکس می‌شود

#### 4. استقلال کامل (Self-Contained)
- **تبدیل به فایل EXE واحد**: قابل اجرا روی هر سیستم ویندوزی بدون نصب پایتون یا کتابخانه
- **دیتابیس داخلی SQLite**: ذخیره تاریخچه تماس‌ها، لاگ‌ها و وعده‌ها به صورت локал
- **موتورهای آفلاین TTS/STT**: قابلیت کارکرد بدون اینترنت (با مدل‌های تعبیه شده)

### 🚀 راهنمای نصب و اجرا

#### مرحله 1: آماده‌سازی محیط (فقط برای توسعه‌دهنده)
قبل از ساخت فایل EXE، باید کتابخانه‌های مورد نیاز نصب شوند:

```bash
pip install requests customtkinter pyinstaller
```

#### مرحله 2: ساخت فایل اجرایی (EXE)
برای تبدیل کد پایتون به یک فایل exe مستقل، دستور زیر را اجرا کنید:

```bash
python build_exe.py
```
پس از اتمام موفقیت‌آمیز، فایل `IlamElectricAgent.exe` در پوشه `dist` ایجاد می‌شود.

#### مرحله 3: اجرای عملیاتی
کافی است فایل `exe.` را روی هر سیستم ویندوزی کپی و اجرا کنید. هیچ نصب اضافی لازم نیست.

### 🔌 اتصال به سامانه واقعی برق ایلام

این سیستم طوری طراحی شده که به صورت مستقیم با زیرساخت‌های موجود شرکت متصل شود:

1. **احراز هویت امن**: استفاده از توکن‌های_session-based برای اتصال به `https://eserv.bargh-ilam.ir`
2. **دریافت اطلاعات مشترک**:
   - شناسه اشتراک
   - مبلغ دقیق بدهی (به ریال)
   - تاریخ آخرین قبض
   - وضعیت خطر قطع انشعاب
3. **ثبت اقدامات**:
   - ثبت وعده پرداخت (تاریخ و مبلغ)
   - ثبت نتایج تماس (پاسخگو بود، اشغال، عدم پاسخ)
   - یادداشت‌برداری هوشمند از محتوای مکالمه

> **نکته امنیتی**: در نسخه نهایی، نام کاربری و رمز عبور اتصال به سامانه اسرو باید از طریق فایل پیکربندی رمزنگاری شده یا متغیرهای محیطی سیستم عامل تامین شود و در کد سخت‌کد (Hardcode) نشود.

### 📊 گزارش‌گیری و پایش

تمامی فعالیت‌ها در دیتابیس داخلی (`ilam_electric_db.sqlite`) ذخیره می‌شوند:
- **جدول `call_logs`**: تاریخچه کامل تماس‌ها، مدت زمان، نتیجه و شماره مشترک.
- **جدول `payment_promises`**: لیست وعده‌های دریافتی، مبلغ و تاریخ تعهد شده.

این داده‌ها قابل استخراج برای گزارش‌های مدیریتی در Excel یا PowerBI هستند.
