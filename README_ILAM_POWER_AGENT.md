---

## 📋 معرفی سیستم

این سامانه یک **هوش مصنوعی مکالمه‌گر کاملاً مستقل و درون‌برنامه‌ای** است که به طور تخصصی برای **شرکت توزیع نیروی برق استان ایلام - شهرستان ایلام** طراحی شده است. سیستم بدون نیاز به هیچ سخت‌افزار تلفنی یا سرویس خارجی، با استفاده از پروتکل‌های نرم‌افزاری VoIP (SIP/RTP) و موتورهای پردازش زبان فارسی، وظیفه تماس با مشترکین بدهکار و وصول مطالبات را انجام می‌دهد.

### 🎯 ماموریت اصلی
- **عنوان رسمی:** اپراتور وصول مطالبات بدهکاران مشترکین برق
- **سازمان:** شرکت توزیع نیروی برق استان ایلام / مدیریت توزیع برق شهرستان ایلام
- **منبع داده:** سامانه eserv.bargh-ilam.ir و پایگاه داده محلی
- **وبسایت رسمی:** [www.ilamedc.ir](https://www.ilamedc.ir/)

## 🚀 ویژگی‌های کلیدی

### 1. **استقلال کامل (100% In-Program)**
- ✅无任何 وابستگی به سخت‌افزار تلفن (مودم GSM، کارت تلفن، سانترال)
- ✅ پیاده‌سازی کامل پروتکل SIP به صورت نرم‌افزاری (Software SIP Stack)
- ✅ تمام کدها، کتابخانه‌ها و ماژول‌ها درون برنامه تعبیه شده‌اند

### 2. **هوشمندی در مکالمه فارسی**
- ✅ تشخیص نیت کاربر (Intent Detection) با دقت بالا
- ✅ تطبیق لحن صحبت بر اساس نوع بدهکار (دوستانه، رسمی، قاطع، حقوقی)
- ✅ استفاده از ادبیات رسمی و بومی استان ایلام
- ✅ اشاره به مکان‌های واقعی (بلوار معلم، شهرک بهشتی، زرجاب، فرهنگیان)

### 3. **استراتژی وصول پویا**
| سطح تشدید | شرایط | لحن مکالمه | اقدام |
|-----------|--------|------------|-------|
| 0 | بدهی کم (<1M ریال) | دوستانه | یادآوری ساده |
| 1 | بدهی متوسط + تاخیر | محترمانه | هشدار جریمه |
| 2 | بدهی بالا (>5M ریال) | رسمی و هشداردهنده | پیشنهاد قسط + اخطار قطع |
| 3 | پرونده قضایی / بدهی خیلی سنگین | قاطع و حقوقی | اقدام نهایی قبل از قطع |

### 4. **پایگاه دانش جامع**
- اطلاعات کامل مشترکین (نام، آدرس، مصرف، تاریخچه پرداخت)
- قوانین و مقررات شرکت برق ایلام (ماده 24، جریمه دیرکرد، شرایط قسط‌بندی)
- تاریخچه تماس‌ها و یادداشت‌های هر مکالمه

## 💾 ساختار پایگاه داده

### جدول `customers` (مشترکین)
| فیلد | توضیح | نمونه |
|------|-------|-------|
| customer_id | شناسه اشتراک 10 رقمی | 1402050102 |
| full_name | نام و نام خانوادگی | علی کریمی |
| address | آدرس دقیق در ایلام | ایلام، محله زرجاب، کوچه امام حسین |
| phone_number | شماره تماس متصل به کنتور | 09183001002 |
| debt_amount | مبلغ بدهی به ریال | 15000000 |
| last_bill_date | تاریخ آخرین قبض | 1402/11/15 |
| consumption_avg | میانگین مصرف (kWh) | 600 |
| payment_history | تاریخچه پرداخت (JSON) | ["بدحساب", "اخطار کتبی"] |
| status | وضعیت فعلی | اخطار نهایی |
| zone | منطقه برق‌رسانی | زرجاب |
| last_call_date | تاریخ آخرین تماس سیستم | 2026-06-03 14:02:20 |

- ماده 24: شرایط قطع انشعاب
- شرایط قسط‌بندی بدهی‌های بالای 5 میلیون تومان
- نرخ جریمه دیرکرد (2% ماهیانه)
- ساعات کاری دفتر بلوار معلم

## 🎯 الگوهای مکالمه (Dialogue Patterns)

### سناریو: درخواست قسط‌بندی
```
[ربات]: بدهی معوقه شما به مبلغ 15,000,000 ریال رسیده و در لیست قطع انشعاب قرار دارید.
[مشترک]: آقا من پول ندارم، میشه قسط بندید؟
[ربات]: بله، با توجه به مبلغ بدهی شما، امکان قسط‌بندی تا 4 ماه وجود دارد. 
        اما باید همین امروز برای تنظیم قرارداد به دفتر بلوار معلم مراجعه کنید. 
        آیا می‌توانید فردا تشریف بیاورید؟
```

### سناریو: اعتراض به مبلغ قبض
```
[مشترک]: این قبض اشتباه اومده، مصرف من اینقدر نبوده!
[ربات]: اگر اعتراض به مبلغ قبض دارید، می‌توانید درخواست بازدید کنتور دهید. 
        اما تا زمان تعیین تکلیف، باید حداقل 50 درصد مبلغ قبض را پرداخت کنید تا قطع نشوید. 
        آیا مایلید راهنمایی کنم چطور درخواست بازدید بدهید؟
```

### سناریو: وعده پرداخت
```
[مشترک]: چشم الان واریز می‌کنم.
[ربات]: بسیار عالی. سیستم بانکی ممکن است تا 24 ساعت تاخیر داشته باشد. 
        اگر تا فردا پیامک تایید نیامد، لطفاً فیش واریزی را به دفتر امور مشترکین ببرید.
```
