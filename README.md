# طناز - دستیار صوتی هوشمند فارسی 🎙️

**Tannaz AI Assistant** - یک دستیار صوتی پیشرفته با قابلیت‌های گرافیکی خیره‌کننده و حافظه ماندگار

## ویژگی‌های اصلی ✨

### 🔊 پردازش صدا
- **اتصال Gemini Live API**: جلسه صوتی دوطرفه با تاخیر کم
- **تشخیص گوینده**: تحلیل pitch (65-300 Hz)، spectral centroid، امضای صوتی
- **دو AudioContext مجزا**: ورودی 16kHz و خروجی 24kHz
- **تحلیلگر فرکانس**: نمودار فرکانس زنده و تشخیص pitch

### 🧠 حافظه هوشمند
- **LocalStorage پایدار**: پروفایل‌های چندگانه با اطلاعات کامل
- **دسته‌بندی خاطرات**: وعده‌ها، خاطرات، ترجیحات
- **امتیاز عشق**: سیستم امتیازدهی احساسی
- **پروفایل‌های پیش‌فرض**: سارا و علی به عنوان نمونه

### 🎨 گرافیک و عناصر بصری

#### صفحه لودینگ
- **کره ماه سه‌بعدی**: WebGL با shaderهای سفارشی
- **نورپردازی PBR**: سایه‌زنی حجمی و درخشش لبه‌ای
- **افکت‌های فیزیکی**: نویز FBM برای سطح ناهموار
- **نوار پیشرفت رنگین‌کمانی**: انیمیشن gradient متحرک

#### ویژوالایزر اصلی
- **کره آیکوساهدرال**: Three.js با هندسه پیچیده
- **Shaderهای GLSL**: افکت‌های پس‌پردازش سفارشی
- **بک‌دراپ داینامیک**: ذرات معلق و نورپردازی پویا
- **دوربین پویا**: حرکت نرم و سینمایی

#### رابط کاربری
- **دکمه‌های کربن فایبر**: طراحی سه‌بعدی با افکت‌های فلزی
- **سایدبار گلس‌مورفیک**: بلور شیشه‌ای با شفافیت
- **پنل‌های تعاملی**: انیمیشن‌های smooth و hover effects
- **نشانگرهای احساسی**: نمایش وضعیت احساسی کاربر

### 🎭 طراحی بصری
- **گلس‌مورفیسم**: افکت شیشه‌ای مدرن
- **تم تاریک لوکس**: رنگ‌های بنفش تیره و قرمز مرجانی
- **پالت رنگی**:
  - بنفش تیره: `#1a0b14`
  - قرمز مرجانی: `#ff453a`
  - نارنجی طلایی: `#ff9500`
  - سبز iOS: `#34c759`
- **تایپوگرافی**: فونت وزیرمتن (RTL) + JetBrains Mono

### 🎬 افکت‌های سینمایی
- **وینیت**: تاریکی لبه‌های تصویر
- **گرین فیلم**: بافت نویز آنالوگ
- **انیمیشن ذرات**: سیستم ذرات سه‌بعدی
- **بلور حرکتی**: motion blur طبیعی

## بهینه‌سازی‌های جدید 🚀

### 1. کش AudioContext
جلوگیری از ایجاد مجدد AudioContext برای بهبود عملکرد

### 2. Lazy Loading
بارگذاری تنبل ماژول‌های بزرگ برای کاهش زمان شروع

### 3. Memory Pool
مدیریت کارآمد حافظه با استفاده از object pool

### 4. Debounce & Throttle
کنترل فراخوانی توابع برای جلوگیری از اجرای مکرر

### 5. Performance Monitor
مانیتورینگ FPS، مصرف حافظه و زمان فریم

### 6. Retry Logic
مدیریت خطا با تلاش مجدد خودکار

### 7. Data Compression
فشرده‌سازی داده‌ها با gzip برای ذخیره‌سازی کارآمد

### 8. Connection Monitor
پایش وضعیت اتصال و سرعت اینترنت
